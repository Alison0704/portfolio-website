4N0M4LY - AI Hardware assistant

  • 4N0M4LY is a cloud-accessible IDE that integrates Verilog design, AI-driven code analysis, and real-time waveform visualization into a single web-based workspace.
  • The system utilizes a hybrid architecture that connects a Gradio frontend on Hugging Face to a local MacBook backend via Cloudflare Tunnels for remote Icarus Verilog and Cocotb execution.
  • The project successfully automates the hardware verification loop, enabling seamless remote simulation and logic-level debugging for digital components.
